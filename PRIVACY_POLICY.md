# Nexus Launcher Privacy Policy

**Last updated:** February 25, 2025

This Privacy Policy (hereinafter — "Policy") describes how the Nexus Launcher software developer, seeeroy (hereinafter — "Developer"), collects, uses, stores, and protects users' personal data (hereinafter — "User").

---

### **1. Types of Data Collected**
1.1. At each launch of the Launcher the following data is automatically collected:
- Operating system (OS) name;
- Processor name;
- Amount of RAM;
- Unique user identifier (ID) generated using the [machine-uuid](https://www.npmjs.com/package/machine-uuid) library;
- Launcher version;
- Java versions installed by the Launcher;
- Number of created builds;
- Number of installed versions;
- Versions of software used within the Launcher.

---

### **2. Data Collection Methods**
2.1. Data is collected by the Launcher at startup by sending information to a server over a secure HTTPS connection.

---

### **3. Use of Data**
3.1. Collected data is used for the following purposes:
- Diagnosing user issues: the unique identifier (ID) allows linking hardware data, Launcher and Java versions to a specific user for technical support;
- Receiving feedback from Users;
- Improving the overall user experience;
- Fixing potential bugs and errors in the Launcher.

3.2. Data is not shared with third parties and is used exclusively by the Developer for the purposes listed above.

---

### **4. Legal Bases for Processing Data**
4.1. Data processing is based on:
- User consent provided at the first launch of the Launcher;
- The Developer's legitimate interest in improving the Launcher and providing technical support.

---

### **5. Users' Rights**
5.1. The User has the following rights regarding their data:
- Right to access data sent by the Launcher;
- Right to correct inaccurate data;
- Right to delete their data;
- Right to restrict data processing;
- Right to withdraw consent to data processing.

5.2. To exercise these rights, the User may send a request to any of the Developer's contact addresses listed on the official website.

---

### **6. Data Storage and Protection**
6.1. The Developer takes all necessary measures to protect Users' data from unauthorized access, alteration, disclosure, or destruction.

6.2. Data is stored on servers controlled by the Developer and protected using modern information security measures.

6.3. The retention period for data is **3 years**. After this period, data will be deleted.

---

### **7. International Data Transfer**
7.1. Users' data is stored and processed in the country where the Developer is located. International data transfer does not occur.

---

### **8. Changes to the Policy**
8.1. The Developer reserves the right to amend this Policy unilaterally. Changes take effect upon publication on the Developer's official website.

8.2. Users will be notified of changes via the Nexus Launcher blog (displayed in the Launcher).

8.3. Continued use of the Launcher after changes to the Policy constitutes the User's acceptance of the new terms.

---

### **9. Consent to the Policy**
9.1. At the first launch of the Launcher, the User will be notified of the need to read this Policy and give consent to data processing. Consent is given by clicking the "Agree" button in the popup window.

9.2. The User may withdraw their consent at any time by sending a request to the Developer's contacts and suspending use of the product.

---

### **10. Contact Information**
10.1. For all questions related to data processing, the User may contact the Developer via the contact details provided on the official website.

By using Nexus Launcher, you confirm that you have read and agree to the terms of this Privacy Policy.
